Η εργασία έχει ολοκληρωθεί θεωρητικά σωστά! Το δεύτερο visit για το typecheking έχει απενεργοποιηθεί.

Με την εντολή java Main.java input1 input2 δημιουργεί 2 output,spg file τα οποια έχουν τον αντίστοιχο "μετατροποιημενο" κώδικα σε spiglet.

Το 3ο visit για την παραγωγή του κώδικα σε spiglet γίνετε με βάση την συνάρτηση generate. Οι εκφράσεις statement εκτυπώνουν το αντίστοιχο κομμάτι τους και οι εκφράσεις exprasion το ίδιο αλλα επιστρέφουν και μια τιμή η οποια ανάλογα με την περίπτωση μπορεί να είναι Integer, κάποια μεταβλητή temp από προηγούμενο expresion, η κάποιο Identifier που τότε θα πρέπει να εργαστούμε ανάλογος.

Για την δημιουργία του v table χρησιμοποιώ στοίβα η οποια γεμίζει με τις συναρτήσεις της κλάσης με την σειρά πou ορίστηκαν και μετά με τις συναρτήσεις των γονέων ώστε όταν θα γεμίσω το vtable να εισαχθούν πρώτα αυτές.
